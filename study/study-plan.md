# Study Plan 스터디 계획

## 목표

스스로 안드로이드 앱 제작 및 예제 코드 분석 가능하도록

## 진도

0. 개발환경설정
    * Android Studio
1. 버전관리 툴 익히기
    * [git](../git/guide.md)
2. Java언어 익히기
    1. 변수 및 데이터 타입(int, float, string, object 등)
        * 사칙연산 연습하면 좋을 듯(a++, ++a a*(b+c) 등 산술연산 순서연습)
    2. 제어문
        * if
        * switch
        * (삼항연산자)
    3. 반복문
        * for
        * while
    4. 객체지향적 언어
        * 4대 개념 짚고, class연습
3. 안드로이드
    1. 안드로이드 개발자 포럼 예제 소스코드 분석
    2. 블루투스 API 사용
    3. 자이로스코프 센서 사용
